# Karesz kijut II.
Karesz azt a feladatot kapta, hogy takarítsa ki a csatornarendszert. 
Ez most egy bonyolultabb csatornarendszer, mert tele van elágazásokkal és több ponton kivezet a csatornarendszerből.
Karesz feladata, hogy a csatornarendszerben lévő összes szemetet összegyűjtse és a végén oda álljon vissza és arra nézzen, ahonnan indult.

Ebben a feladatban **nem lehet ultrahang szenzorokat használni!**


# K.A.R.E.SZ.
A Középiskolai Algoritmusokra Rávezető Edukációs Szoftver (továbbiakban K.A.R.E.SZ.) 
a [Karel nyelv](https://en.wikipedia.org/wiki/Karel_(programming_language) egy C# nyelven történő implementációja, 
amely párhuzamos programozási eszközökkel teszi lehetővé a programozás középiskolai oktatását. 
A Kőbányai Szent László Gimnázium informatika tagozatán való alkalmazása Nagy Tibor munkássága óta hagyomány.
